Tela Inicial

Após o login essa tela aparecera, nela o usuario tera acesso.
Cadastro cliente: Cadatrar novos clientes e gerando uma nova conta.
Buscar clientes: Acesso a clientes já cadastrados no banco de dados.
Simular emprestimos: Simulação de emprestimos para compra de casas ou carros.

Tela Cadastro

Nesta tela todos os campos devem ser preechidos para que este cliente seja valido, somente a agencia e conta serão preechidos automaticamente, com base em clientes já existentes. CPF é validado de forma imediata após ser preenchido e seu feedback é dado ao lado. Dentro do elemento endereço temos duas opções para preecher.
1ª Atraves do cep o sistema auto completa Rua, Cidade, Bairro e UF(Unidade Federativa)
2ª Preenchimento manual de todas as informações.

Tela pesquisa clientes

Neste sistema existem duas maneiras de pesquisa de clientes. A primeira atraves do CPF do mesmo, a segunda atraves de uma lista com todos os clientes da base ordenados em ordem alfabetica.

Tela cliente

É possivel nesta tela alterar todos os dados mutaveis do cliente, como nome, telefone e endereço mas não é possivel alteração em cpf e data de nascimento por exemplo pois são dados que não se alteram. Além destas alterações é possivel excluir o cliente do banco de dados.

Tela Simulação

Nesta tela é realizada uma simulação calculando o salario que foi informado no cadastro, é realizada em pró de não comprometer 25% da renda mensal deste cliente e o feedback é mostrado, de forma instantanea
obs: clientes sem cadastro não são aprovados.
